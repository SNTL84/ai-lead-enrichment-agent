# 🤝 Contributing to AI Lead Enrichment Agent

Thank you! This project welcomes contributions from developers, sales automation engineers, and growth hackers.

---

## 📋 How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/SNTL84/ai-lead-enrichment-agent.git
cd ai-lead-enrichment-agent
```

### 2. Create a Feature Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Test Locally
Open the HTML file directly in any browser — no build step required.

### 4. Commit & Push
```bash
git commit -m "feat: describe your change"
git push origin feature/your-feature-name
```

### 5. Open a Pull Request
